# 한국 학원 SaaS 경쟁 조사

조사일: 2026-05-06

목적: Academy Follow-up이 정면으로 붙어야 할 시장과 피해야 할 시장을 구분한다. 공개 가격과 공식 페이지에서 확인 가능한 기능만 기준으로 정리했다.

## 핵심 결론

현재 한국 학원 SaaS는 대부분 "전체 학원관리"를 판다. 출결, 원생/반/강사 관리, 수납, 상담, 학부모 앱, 알림톡/문자, 리포트, AI 기능을 묶어서 월 구독으로 제공한다.

Academy Follow-up이 처음부터 이들과 같은 폭으로 경쟁하면 기능 양에서 밀린다. 대신 첫 포지션은 다음처럼 좁히는 편이 낫다.

> 선생님이 수업 직후 결석, 재시험, 숙제 미완료 같은 후속 연락을 10초 안에 처리하고 기록하는 모바일 우선 도구.

## 가격 정책 패턴

1. 월 구독은 대체로 5,900원에서 159,000원 사이에 공개되어 있다.
2. 소규모 학원용은 9,900원에서 33,000원 사이가 많다.
3. 100명 전후 학원은 29,000원에서 99,000원 사이로 갈린다.
4. 문자/알림톡은 월 구독과 별도로 건당 과금하거나 크레딧 충전한다.
5. 무료 체험은 14일 또는 30일이 흔하다.
6. 한국 학원 SaaS는 카드 등록 없는 체험, 카카오 상담, 데이터 이전, 1:1 세팅 지원을 자주 내세운다.

## UI 패턴

1. 원장 화면은 대시보드형이다. 오늘 수업, 출석, 미납, 문의, 리포트 발송 대기 같은 운영 지표를 보여준다.
2. 선생님 화면은 모바일 앱 또는 모바일 웹 액션형이다. 출결, 문자, 상담, 수업 기록을 빠르게 처리하는 흐름이 많다.
3. 학부모/학생 앱을 별도로 두는 제품이 많다. 다만 이 구조는 초기 구현과 운영 부담이 크다.
4. 랜딩 페이지는 기능 설명보다 실제 화면 예시, 숫자, 가격, 무료 체험 CTA를 앞에 둔다.
5. "AI가 다 해준다" 포지션이 늘고 있지만, 실제 현장에서는 선생님이 문구를 확인하고 보내는 안전장치가 중요하다.

## 우리 제품 방향

초기 포지션:

> 학원관리 프로그램을 바꾸지 않고도, 수업 직후 학부모 후속 안내를 빠르게 보내고 기록하는 선생님용 도구.

초기 가격 가설:

| 플랜 | 가격 | 기준 |
| --- | --- | --- |
| Pilot | 1개월 무료 | 첫 학원 도입, 학생/선생님 데이터 세팅 포함 |
| Basic | 29,000원/월 | 학생 100명까지, 문자/알림톡 실비 별도 |
| Standard | 49,000원/월 | 학생 300명까지, 선생님 20명까지, 템플릿/기록/권한 포함 |

친구 학원 규모가 학생 200명, 선생님 16명 정도라면 Standard 49,000원/월이 첫 유료 기준으로 적당하다. 단, 처음에는 돈보다 실제 사용 빈도 검증이 더 중요하다.

## 다음 제품 결정

다음 개발 우선순위는 전체 학원관리 기능이 아니라 아래 순서가 맞다.

1. 선생님이 수정한 문자를 실제 후속 기록으로 저장한다.
2. 발송 전 dry-run/테스트 발송을 제공한다.
3. 학생/선생님/반 관리 화면은 최소 CRUD로 붙인다.
4. 원장용 홈은 "오늘 미처리 후속", "이번 주 후속 건수", "선생님별 처리 상태" 중심으로 만든다.
5. 결제/수납/학부모 앱/AI 리포트는 초반 범위에서 제외한다.
