## 확인 기준

- 전체문자는 `전체 학생`, `반 선택`, `학년 선택` 대상 범위를 지원해야 합니다.
- `중복 수신자 제외`가 켜져 있으면 같은 전화번호는 한 번만 발송 후보에 남아야 합니다.
- 신규 선생님 등록 시 이메일 입력이 보이지 않아야 합니다.
- 반 등록/수정 시 학년은 선택형이어야 합니다.
- 랜딩 페이지는 현재 구현된 모바일 출석, 지각/결석 문자, 전체문자, 학생/반/스케줄 관리 가치를 설명해야 합니다.
