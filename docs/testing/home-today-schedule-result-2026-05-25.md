# T-431 로그인 홈 오늘 일정판 결과 - 2026-05-25

## 반영 내용

- 로그인 후 홈에 `오늘 일정` 섹션을 추가했습니다.
- 기존 모바일 홈 흐름은 `히어로 -> 날짜 선택 -> 오늘 일정 -> 빠른 시작 -> 오늘 요약`으로 유지했습니다.
- 우리 학원 수업은 반/시간 기준으로 묶어 `출석 보기` 액션을 제공합니다.
- 상담, 외부 일정, 학생별 일정은 읽기 전용 일정으로 표시합니다.
- 타 학원 공유 일정은 실제 학원명 없이 `연결 학원 일정`으로 표시합니다.

## 권한 기준

- 원장/관리자는 전체 학원 일정 상세와 공유 일정을 볼 수 있습니다.
- 선생님/보조 선생님은 담당 반/학생 일정 상세만 볼 수 있습니다.
- 선생님/보조 선생님도 홈 상단 요약에서는 전체 학원의 수업 수와 공유 일정 수를 확인할 수 있습니다.

## 확인 항목

- 모바일 390px에서 오늘 일정 섹션이 날짜 선택 아래에 표시됩니다.
- 우리 학원 수업 행을 누르면 출석 탭으로 이동합니다.
- 공유 일정 행은 이동하지 않고 읽기 전용으로 표시됩니다.
- 연결 학원명과 전화번호는 노출되지 않습니다.
- 일정이 없는 날짜에는 `이 날짜에는 표시할 일정이 없습니다.`가 표시됩니다.

## 후속 후보

- 주간 캘린더형 보기
- 수업 행 클릭 시 출석 탭에서 해당 수업까지 자동 선택
- 선생님별 담당 일정 필터를 홈에서 직접 변경하는 기능
