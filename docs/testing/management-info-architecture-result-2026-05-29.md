# 관리 화면 정보 구조 정리 결과 - 2026-05-29

## 범위

- T-443 관리 화면 B2B 정보 구조 정리
- 브랜치: `codex/t-443-management-info-architecture`
- 백업 브랜치: `backup/main-before-management-ia-2026-05-29`

## 변경 내용

- 관리 화면 진입부를 업무 그룹형 메뉴로 정리했습니다.
- 기존 `원장 시작 / 학생 / 반 / 구성원 / 문자 / 설정` 기능은 유지하되, 사용자에게는 아래 업무 단위로 보이게 했습니다.
  - 시작: 초기 운영 순서
  - 명단: 학생, CSV, 스케줄, 공유 동의
  - 수업: 반, 담당 선생님, 반 공통 수업 시간
  - 직원: 직위, 권한, 담당 반
  - 문자: 사유별 템플릿
  - 정책: 발신 정보, dry-run, 보조 선생님 발송 권한
- 메뉴마다 현재 상태를 짧게 표시합니다.
  - 학생 수
  - 스케줄 미등록 학생 수
  - 반 수
  - 직원 수
  - 템플릿 수
  - 테스트 발송/실발송 상태

## 유지한 내용

- 학생/반/구성원/템플릿/설정 저장 API는 변경하지 않았습니다.
- Supabase schema 변경은 없습니다.
- 원장/관리자만 관리 탭에 접근하는 권한 구조는 유지합니다.

## 확인 기준

- 모바일 390px에서 관리 탭 첫 화면이 업무 단위 메뉴로 보입니다.
- `시작 / 명단 / 수업 / 직원 / 문자 / 정책` 메뉴를 눌러 기존 섹션으로 이동할 수 있습니다.
- 기존 학생 등록, CSV 등록, 반 등록, 수업 시간 등록, 구성원 등록, 템플릿 저장, 설정 저장 흐름이 유지되어야 합니다.
