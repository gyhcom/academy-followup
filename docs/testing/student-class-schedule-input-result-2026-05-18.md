# 학생/반 스케줄 입력 UX 보강 결과

날짜: 2026-05-18

## 범위

- T-419 학생 등록 시 학교/학년/반 선택 흐름 개선
- T-420 반 생성 및 수업 요일 묶음 등록 개선

## 변경 내용

- 학생 등록/수정 폼의 학년 입력을 자유 입력에서 선택형으로 변경했습니다.
- 선택값은 `무학년제`, `초등`, `중1`, `중2`, `중3`, `고1`, `고2`, `고3`, `기타`입니다.
- 반 공통 스케줄 등록 폼에서 요일을 여러 개 선택할 수 있게 했습니다.
- 빠른 선택으로 `월수금`, `화목`, `월수`를 제공합니다.
- 직접 요일 버튼을 눌러 원하는 조합도 만들 수 있습니다.
- 반 공통 스케줄 API는 새 `dayOfWeeks` 배열 요청을 처리합니다.
- 기존 `dayOfWeek` 단일 요청도 계속 처리해 하위 호환성을 유지했습니다.
- 중복 판단은 학생 ID, 요일, 시간, 수업 유형 기준으로 처리합니다.

## 확인 기준

- 원장이 반 전체 학생에게 월/수 수업을 한 번에 등록할 수 있어야 합니다.
- 화/목, 월/수/금 수업도 한 번의 입력으로 등록할 수 있어야 합니다.
- 이미 같은 요일/시간 스케줄이 있는 학생은 중복 생성하지 않아야 합니다.
- 학생 등록 시 학년을 선택형으로 고를 수 있어야 합니다.

## 남은 확인

- 실제 Supabase 데이터에서 반 전체 학생 30명 이상 기준으로 월수/화목 등록 결과를 확인해야 합니다.
- 모바일 390px 관리 화면에서 요일 버튼이 과하게 밀리지 않는지 추가 확인이 필요합니다.
