# 홈 공유 일정 가시성 개선 결과 - 2026-05-27

## 목적

홈 `오늘 일정`에 타 학원/연결 학원 일정이 들어와도 원장이 공유 기능이 동작하는지 알아보기 어려운 문제를 보정했습니다.

## 반영 내용

- 홈 히어로 보조 요약을 `오늘 수업 N개 · 타 학원 M건 · 연결 공유 K건`으로 변경했습니다.
- 오늘 일정 섹션 우측 요약을 `수업 / 타학원 / 연결`로 분리했습니다.
- 타 학원 또는 연결 공유 일정이 있으면 `보강 불가 일정이 있습니다` 안내를 일정 목록 상단에 표시합니다.
- 수동 등록 타 학원 수업은 `타 학원 수업` 배지로 표시합니다.
- SaaS 학원 간 자동 공유 일정은 `연결 공유` 배지와 `익명 공유` 문구로 표시합니다.
- 실제 상대 학원명과 전화번호는 계속 노출하지 않습니다.

## 확인 순서

1. 관리 탭에서 `중2 수학 A반`, `중3 영어 B반`을 만듭니다.
2. 위 CSV를 일괄 등록합니다.
3. 반 공통 수업 시간을 등록합니다.
4. 학생 상세에서 일부 학생에게 `타 학원 수업`을 연결합니다.
5. 다른 학원 테스트 계정에 같은 학생을 등록하고 동의 조건을 맞춰 자동 공유를 확인합니다.
6. 홈에서 `오늘 수업`, `타 학원`, `연결 공유` 숫자가 분리되어 보이는지 확인합니다.
7. 오늘 일정 상단에 `보강 불가 일정이 있습니다` 안내가 보이는지 확인합니다.

## 운영형 공유 일정 데모 Seed

한 명짜리 공유 테스트가 아니라 실제 운영처럼 여러 학생이 여러 학원 일정과 연결된 상태를 보려면 아래 순서로 실행합니다.

1. Supabase SQL Editor에서 `supabase/seed-demo-operations.sql`을 먼저 실행합니다.
2. 이어서 `supabase/seed-shared-schedule-operations-demo.sql`을 실행합니다.
3. SQL 결과에서 아래 count를 확인합니다.
   - `shared_demo_academy_count`: 3
   - `local_consent_on_count`: 10명 이상
   - `remote_consent_on_count`: 10명 이상
   - `active_shared_link_count`: 10건 이상
   - `remote_shared_schedule_count`: 15건 이상
4. 더배움 원장 계정으로 로그인해 홈 날짜를 월~토로 바꿔봅니다.
5. 공유 일정이 있는 날짜에서 아래 표현을 확인합니다.
   - `연결 공유 N건`
   - `보강 불가 일정이 있습니다`
   - `학생명 · 연결 공유 · 익명 공유 · 보강 불가`

운영형 공유 seed는 아래 타 학원 더미를 만듭니다.

- 다솜수학학원
- 리딩논술학원
- 탑클래스영어학원

이 seed의 목적은 SaaS 연결 학원 일정이 홈에서 잘 드러나는지 확인하는 것입니다. 실제 상대 학원명과 전화번호는 더배움 홈에 노출하지 않습니다.

## 남은 확인

- 실제 친구 학원 반명/학생명/스케줄 20~30명 투입은 T-501에서 진행합니다.
- 실제 SMS 발송은 이번 범위가 아니며 `sms_dry_run=true`를 유지합니다.
