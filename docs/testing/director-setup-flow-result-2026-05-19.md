# 원장 시작 순서 고도화 결과

날짜: 2026-05-19

## 원장에게 전달할 완료 내용

- 관리 탭 첫 화면을 `원장 시작 순서`로 정리했습니다.
- 시작 순서는 `선생님 등록 -> 반 만들기 -> 학생 등록 -> 수업 시간 등록 -> 출석 확인`입니다.
- 각 단계에서 현재 상태를 숫자로 볼 수 있습니다.
- 각 버튼을 누르면 해당 입력 화면 또는 출석 화면으로 바로 이동합니다.
- 반/학생 학년 선택에 `무학년제` 옵션을 반영했습니다.

## 구현 내용

- 초기 세팅 섹션 이름을 `원장 시작 순서`로 변경했습니다.
- 단계 CTA 문구를 원장이 이해하기 쉬운 말로 바꿨습니다.
- `출석 확인` 단계와 CTA를 추가하고 출석 탭으로 연결했습니다.
- 오늘 출석 기록 기준 수업 수를 관리 화면에 전달해 표시합니다.
- 기존 `무학년` 표기는 폼과 화면에서 `무학년제`로 보이도록 정리했습니다.

## 확인 기준

- 원장 계정으로 관리 탭에 들어가면 시작 순서가 바로 보여야 합니다.
- `선생님 등록`, `반 만들기`, `학생 등록`, `수업 시간 등록`, `출석 확인` 버튼이 각각 동작해야 합니다.
- 반/학생 등록 폼에서 `무학년제`를 선택할 수 있어야 합니다.
- 모바일 390px에서 단계 카드가 가로로 밀리지 않아야 합니다.
