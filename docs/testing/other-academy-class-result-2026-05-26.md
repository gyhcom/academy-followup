# T-432 타 학원 수업 일정 모델/입력/홈 표시 결과

## 구현 범위

- 새 테이블:
  - `external_academies`
  - `external_academy_classes`
  - `student_external_class_enrollments`
- 학생 상세 `타 학원 수업` 섹션 추가
- 타 학원 수업 등록/학생 연결/연결 해제 API 추가
- 홈 오늘 일정판에 수동 타 학원 수업 표시
- 보강 후보 충돌 계산에 수동 타 학원 수업 포함
- 기존 `external` 일정 라벨을 `개인/기타 일정`으로 변경

## 확인 기준

- 원장 계정에서 학생 상세에 들어간다.
- `타 학원 수업` 섹션에서 `논술학원`, `논술 정규반`, `화 19:30-20:30`을 등록한다.
- 홈 오늘 일정에서 해당 요일에 `타 학원 수업`이 시간순으로 표시되는지 확인한다.
- 문자/보강 후보에서 같은 시간대가 충돌로 막히는지 확인한다.
- `연결 학원 수업`은 계속 실제 학원명 없이 익명으로 표시되는지 확인한다.

## 운영 메모

- 이 기능은 SaaS 미사용 학원을 위한 수동 일정입니다.
- SaaS 사용 학원끼리는 기존 동의 기반 자동 공유를 유지합니다.
- 기존 `개인/기타 일정`은 메모성 일정으로 남깁니다.
