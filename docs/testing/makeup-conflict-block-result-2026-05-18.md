# 보강 불가 시간 표시 강화 결과

날짜: 2026-05-18

## 원장 요구

- 학생 기존 수업 시간이 16:30-19:00이면 그 시간으로 보강 안내가 나가면 안 됨
- 기존 스케줄 카드의 보강 안내 버튼은 헷갈리므로 제거해도 됨

## 변경 내용

- 보강 달력에서 선택한 날짜/시간이 기존 일정과 겹치면 보강 후보 선택 버튼을 비활성화했습니다.
- 비활성 버튼 문구는 `기존 일정과 겹쳐 선택 불가`로 표시합니다.
- 겹치는 일정의 제목, 시간, 일정 유형을 경고 카드에 표시합니다.
- 주간 스케줄 카드에서는 기존 일정의 `이 시간으로 보강 안내` 버튼을 제거했습니다.
- 기존 일정 카드는 `이미 등록된 일정입니다. 이 시간은 보강 후보에서 제외합니다.` 안내만 보여줍니다.

## 확인 기준

- 기존 수업/외부 일정/보강/상담과 겹치는 시간은 보강 후보로 선택할 수 없어야 합니다.
- 겹치지 않는 시간은 기존처럼 보강 후보로 선택할 수 있어야 합니다.
- 문자 미리보기, dry-run 발송, 발송 후 보강 스케줄 등록 흐름은 기존처럼 유지되어야 합니다.

## 남은 확인

- 실제 운영형 demo seed 기준으로 학생별 겹침 케이스를 더 많이 넣어 모바일에서 확인할 필요가 있습니다.
