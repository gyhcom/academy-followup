# T-445 감사 로그 + 최근 변경 이력 결과

## 범위

- `audit_logs` 테이블을 추가해 운영 데이터 변경 이력을 남긴다.
- 관리 탭에 `이력` 메뉴를 추가해 최근 20건을 읽기 전용으로 보여준다.
- 전화번호 원문은 감사 로그 summary에 남기지 않는다.

## 반영 항목

- 학생 생성/수정
- 반 생성/수정
- 학생 스케줄 생성/수정/삭제
- 반 공통 수업 시간 일괄 등록
- 타 학원 수업 연결/해제
- 문자 템플릿 수정
- 학원 운영 설정 수정

## 확인 기준

- 원장/관리자는 관리 탭 `이력`에서 최근 변경 이력을 확인할 수 있다.
- 선생님/보조 선생님은 관리 탭 접근 제한으로 이력을 볼 수 없다.
- 감사 로그 저장 실패는 기존 저장 작업을 막지 않는다.
- 실제 문자/출석 기록은 기존 `message_logs`, `followups`, `attendance_records`를 기준으로 본다.

## 운영 DB 적용

Supabase SQL Editor에서 아래 migration을 실행해야 운영 DB에 이력 저장이 시작된다.
