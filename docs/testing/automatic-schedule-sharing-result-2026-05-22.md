# T-429 동의 기반 학원 간 스케줄 자동 공유 결과

## 결과

- 학생 등록/수정 저장 후 동의가 확인된 학생은 같은 SaaS 안의 다른 학원 학생과 자동 매칭됩니다.
- 자동 매칭 기준은 이름, 학교, 학년, 전화번호입니다.
- 전화번호는 학부모 연락처와 학생 연락처 중 하나라도 일치하면 같은 학생 후보로 봅니다.
- 동의 체크를 끄거나 학생 정보가 달라지면 기존 active 공유 연결을 해제합니다.
- 자동 연결 후에도 실제 학원명과 전화번호는 상대 학원 화면/API 응답에 노출하지 않습니다.
- 기존 공유 코드 방식은 자동 연결이 되지 않는 예외 상황의 보조 기능으로 유지합니다.

## 확인 기준

- 더배움 김민준과 다솜 김민준의 이름/학교/학년/전화번호/동의 체크가 같으면 저장 후 자동 연결됩니다.
- 둘 중 한쪽이라도 동의 체크가 없으면 자동 연결되지 않습니다.
- 이름/학교/학년이 같아도 전화번호가 다르면 자동 연결되지 않습니다.
- 동의 체크를 해제하면 양쪽 학생 상세에서 공유 스케줄이 사라집니다.
- 학생 상세에는 실제 학원명 대신 `연결 학원 1`과 바쁜 시간대만 표시됩니다.

## 운영 메모

- 연결 전에는 다른 학원 존재 여부나 후보 수를 보여주지 않습니다.
- 전화번호는 서버 내부 동일 학생 검증에만 사용합니다.
- 파일럿 동의 방식은 종이 동의서 수령 후 원장 수동 체크입니다.
