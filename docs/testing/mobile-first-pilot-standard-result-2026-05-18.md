# 모바일 우선 파일럿 UX 기준 전환 결과

날짜: 2026-05-18

## 목적

파일럿 MVP의 제품 기준을 PC 대시보드가 아니라 모바일 운영 앱으로 고정했습니다.

실제 사용자는 선생님과 원장이 수업 중 또는 수업 직후 핸드폰으로 접속하는 경우가 많으므로, 앞으로 핵심 흐름은 모바일 390px 화면을 먼저 통과해야 합니다.

## 적용 기준

- 1순위 화면: 모바일 390px
- 핵심 흐름: 홈, 출석, 문자, 관리
- 보조 화면: iPad, PC
- PC 기준: 가로 overflow 없음, 주요 버튼/입력 가능, 탭 접근 가능
- 보류 기준: PC 전용 고급 대시보드 완성도, 넓은 화면 전용 정보 밀도 개선

## 화면별 확인 방향

- 홈: 오늘 할 일, 운영 기준 날짜, 요약 숫자가 첫 화면에서 바로 이해되어야 합니다.
- 출석: 수업 선택 후 학생 도착 토글이 빠르게 보여야 합니다.
- 문자: 학생 선택 후 모바일 하단 CTA 또는 바텀시트로 초안 작성이 바로 이어져야 합니다.
- 관리: 초기 세팅, 학생/반/스케줄 입력이 모바일에서 막히지 않아야 합니다.
