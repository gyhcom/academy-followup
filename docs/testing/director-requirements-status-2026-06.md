# 원장 요구사항 완료/미완료 정리 - 2026-06-26

이 문서는 원장님에게 현재 진행 상태를 공유하기 위한 기준입니다.

상태는 `완료`, `부분 완료`, `미완료`로 구분합니다.

## 1. 카톡 복사용 요약

원장님 요청사항 기준으로 현재 상태 정리드립니다.

완료된 부분

- 메뉴 순서는 `오늘 -> 출석부 -> 문자 -> 리포트 -> 관리` 흐름으로 정리했고, 출석부가 두 번째에 오도록 했습니다.
- 출석부는 ON/OFF 방식이 아니라 `출석 / 지각 / 결석 / 미체크 / 확인 필요` 중심으로 정리했습니다.
- 출석부에서 지각/결석 학생을 여러 명 선택해 한 번에 문자 기록 저장/테스트 발송하는 흐름을 만들었습니다.
- 문자 본문에서 학생 이름은 `{{studentName}}` 변수로 학생별 자동 치환됩니다.
- 날짜 기준 달력형 출석부를 만들었고, 날짜를 누르면 해당 날짜의 수업/학생/연락 필요 목록을 확인할 수 있습니다.
- 학생을 클릭하면 학생 정보, 선택 날짜 수업, 주간 스케줄, 최근 연락 기록을 볼 수 있습니다.
- 동명이인은 학교/학년/반/마스킹 연락처로 구분할 수 있게 했습니다.
- 학생 등록 CSV 템플릿은 Excel에서 한글이 깨지지 않도록 보정했습니다.
- 재원생 삭제는 기록 보존 문제 때문에 삭제 대신 `퇴원` 상태로 관리하도록 화면에 안내를 추가했습니다.

부분 완료된 부분

- 지각/결석 다중 선택 문자는 현재 `기록 저장`과 `테스트 발송` 흐름까지 됩니다. 실제 학부모 대상 문자 발송은 SOLAPI 테스트 번호 1건 검증 후 켜는 것이 안전합니다.
- 학생 기준 출석 장부는 현재 날짜/수업 기준 확인은 가능하지만, 학생 row와 날짜 column이 있는 월간 matrix 장부는 아직 별도 구현 전입니다.
- 학생 개인 메모는 연락 기록/문자 기록 중심으로 남길 수 있지만, 별도 개인 메모장 기능은 아직 없습니다.

아직 미완료인 부분

- 수강료, 교재비, 기타 비용 변수를 관리자가 설정하는 기능
- 교재 선택에 따른 월말 교재비/수강료 합산 문자
- 한 반 전체를 한 번에 출석 처리하는 기능
- 성별 기반 아이콘 구분
- 학생 기준 월간 출석 matrix
- 학생 개인 메모장

현재 기준으로는 `출석부 확인 -> 지각/결석 학생 선택 -> 학생명 자동 치환 문자 확인 -> 기록 저장/테스트 발송` 데모는 가능합니다.

다음 큰 결정은 실제 운영 데이터 20~30명을 넣고 테스트 번호 1건으로 실제 문자 수신을 확인할지 여부입니다.
