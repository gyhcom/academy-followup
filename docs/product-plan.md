# 제품 기획서

## 1. 제품 정의

**Academy Follow-up**은 한국 학원 선생님이 수업 직후 학생별 후속 안내를 빠르게 처리하고, 원장이 전체 팔로업 현황을 확인할 수 있게 하는 학원 운영 SaaS입니다.

첫 버전은 거대한 학원관리시스템이 아닙니다. 핵심은 아래 한 가지입니다.

> 학생 이름 클릭 -> 사유 선택 -> 학부모 문자 미리보기 -> 발송 -> 기록 저장

## 2. 파일럿 학원

- 학원명: 더배움프라임영수학원
- 업종: 학원 등 교습시설
- 주소: 충남 아산시 탕정면 한들물빛5로 17 트윈시티2 603호, 604호
- 위치: 탕정역 1번 출구에서 187m
- 영업시간: 23:00 운영 종료
- 네이버 플레이스 ID: 1298554109
- 방문자 리뷰: 16개
- 블로그 리뷰: 29개
- 예상 규모: 원생 약 200명, 선생님 약 16명

## 3. 문제

원생 200명 규모의 학원에서는 결석, 지각, 숙제 미완료, 재시험, 상담 권장 같은 후속 안내가 매일 반복됩니다.

현재 이 업무가 카톡, 문자, 엑셀, 선생님 개인 메모에 흩어지면 아래 문제가 생깁니다.

- 선생님마다 안내 기준과 말투가 달라짐
- 원장이 누가 어떤 학부모에게 연락했는지 알기 어려움
- 같은 학생에게 중복 발송하거나 꼭 보내야 할 안내가 누락됨
- 학부모 연락 기록이 학생별로 쌓이지 않음
- 보조 선생님이 체크한 내용이 정규 선생님/원장에게 늦게 전달됨
- 학부모 상담 전에 과거 연락 이력을 다시 찾기 어려움

## 4. 제품 가설

선생님이 수업 직후 10초 안에 학생별 팔로업 문자를 만들고 발송할 수 있다면, 학부모 안내 누락이 줄고 학원 내부 운영 기록이 정리됩니다.

원장에게 중요한 가치는 문자 발송 자체가 아니라 아래입니다.

- 오늘 어떤 학생에게 연락했는지 확인
- 반별/선생님별 팔로업 누락 확인
- 학생별 학부모 연락 히스토리 축적
- 선생님별 안내 문구와 기준 통일
- 상담 전 학생 상태 파악 시간 단축

## 5. 사용자

### 원장

- 전체 학원 현황을 봅니다.
- 발송 기록, 반별 현황, 선생님별 처리 여부를 확인합니다.
- 문자 템플릿과 발신 설정을 관리합니다.

### 정규 선생님

- 담당 반 학생의 결석, 재시험, 숙제 미완료 등을 학부모에게 안내합니다.
- 학생별 팔로업 기록을 확인합니다.

### 보조 선생님

- 출석/숙제/재시험 대상 같은 수업 후 체크를 빠르게 남깁니다.
- 학원 설정에 따라 직접 발송하거나 초안만 작성합니다.

표현 원칙은 학생을 낙인찍지 않는 것입니다. "위험 학생" 같은 표현은 쓰지 않고, "팔로업 필요", "확인 필요", "상담 권장"처럼 다음 행동 중심의 표현을 씁니다.

## 9. 문자 템플릿

결석:

```text
[{{학원명}}] 안녕하세요. 오늘 {{학생명}} 학생이 수업에 결석하여 안내드립니다.
확인 부탁드리며, 보강 일정이 필요한 경우 담당 선생님이 다시 안내드리겠습니다.
```

재시험:

```text
[{{학원명}}] 안녕하세요. {{학생명}} 학생은 오늘 수업 내용 중 재시험이 필요하여 안내드립니다.
다음 수업 전까지 오답을 다시 확인할 수 있도록 지도 부탁드립니다.
```

숙제 미완료:

```text
[{{학원명}}] 안녕하세요. {{학생명}} 학생이 이번 과제를 완료하지 못해 안내드립니다.
다음 수업 전까지 과제를 마무리할 수 있도록 확인 부탁드립니다.
```

상담 권장:

```text
[{{학원명}}] 안녕하세요. {{학생명}} 학생의 최근 학습 흐름에 대해 간단한 상담이 필요하여 안내드립니다.
가능하신 시간에 담당 선생님에게 연락 부탁드립니다.
```

## 10. 학원별 브랜딩

이 서비스는 여러 학원이 쓰는 SaaS가 되어야 하지만, 로그인 후 화면에서는 서비스 브랜드보다 학원 브랜드가 먼저 보여야 합니다.

### 원칙

- 상단에는 학원명과 로고를 보여줍니다.
- 주요 문구는 "대시보드"보다 "운영 보드", "오늘의 팔로업"처럼 내부 운영 언어를 씁니다.
- 문자 미리보기에는 SaaS 이름이 아니라 학원 발신명이 들어갑니다.
- 학원별 템플릿, 발신번호, 선생님명 표시 방식을 따로 관리합니다.

### 초기 커스터마이징

- 학원명
- 학원 slug
- 네이버 플레이스 ID
- 주소
- 발신명
- 발신번호
- 대표 색상
- 문자 템플릿

### 나중 확장

- 로고
- 커스텀 도메인
- 카카오 알림톡 채널
- 지점 관리
- 학부모용 리포트 브랜딩

## 11. 성공 기준

파일럿 2주 기준으로 아래를 봅니다.

- 선생님 2명 이상이 실제 수업 후 사용
- 하루 5건 이상 팔로업 기록 생성
- 발송 전 문구 수정 비율 확인
- 중복 발송/오발송 발생 여부 확인
- 원장이 발송 기록 화면을 실제로 확인하는지 확인
- 선생님이 기존 방식보다 빠르다고 느끼는지 확인
- 학부모 반응에서 부정적인 피드백이 없는지 확인

## 12. 확장 로드맵

### 1단계: 문자 팔로업

- 학생 클릭
- 사유 선택
- 문자 발송
- 발송 기록

- 템플릿 관리
- CSV 학생 일괄 등록

### 3단계: 학습 케어 기록

- 출석 체크
- 숙제 체크
- 재시험 완료 여부
- 상담 기록
- 확인 필요 학생 리스트

### 4단계: 학부모 커뮤니케이션

- 주간 학부모 리포트
- 상담 준비 리포트
- 카카오 알림톡
- 학부모용 공유 링크

## 13. 가격 가설

초기 상용화 기준입니다.

- Starter: 월 5만~10만원, 학생 100명 이하
- Standard: 월 15만~25만원, 학생 300명 이하
- Pro: 월 30만원 이상, 다중 지점/고급 리포트

문자 발송 비용은 월 요금에 일부 포함하거나 별도 과금합니다.
