# 기능 현황판

마지막 업데이트: 2026-05-28

이 문서는 긴 티켓 목록을 대신해 현재 제품 기능을 한눈에 보기 위한 요약표입니다.  
상세 구현 범위와 완료 기준은 `docs/planning/tickets.md`를 기준으로 봅니다.

## 상태 기준

- 완료: 현재 main에 반영되어 파일럿에서 사용할 수 있음
- 부분 완료: 핵심은 있으나 스키마/실데이터/운영 검증 또는 일부 기능이 남음
- 예정: 다음 개발 후보
- 보류: 지금 파일럿 범위에서는 하지 않음
- 상용화 이후: 개인정보, 과금, 확장성 때문에 파일럿 이후 판단

## 운영 메모

- 현재 제품 기준은 PC 대시보드가 아니라 모바일 운영 앱입니다.
- PC/iPad는 기능 접근과 화면 깨짐 방지만 확인합니다.
- 실제 문자 발송 전까지는 `sms_dry_run=true`를 유지합니다.
- 신규 기능을 완료하면 이 문서와 `tickets.md`를 함께 갱신합니다.
