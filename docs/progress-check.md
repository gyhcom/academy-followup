# 진행 체크

마지막 업데이트: 2026-05-07

## 현재 제품 방향

Academy Follow-up은 단순 문자 발송 도구가 아니라, 학생별 주간 스케줄과 팔로업 연락을 함께 관리하는 학원 운영 SaaS로 잡습니다.

현재 핵심 문장은 아래입니다.

> 학생별 수업·외부일정·보강 가능 시간을 한눈에 보고, 필요한 학부모 연락을 바로 만드는 학원 운영 보드

- 보강 안내 문자 기본 템플릿
- 보강 후보 시간 변수 치환

## 부분 완료 또는 보류

- 문자 발송 API는 dry-run까지 완료, 실제 SMS 발송은 SOLAPI 환경변수와 `SMS_DRY_RUN=false` 전환 필요
- 반 비활성화는 `classes.status` 컬럼이 필요해 보류
- 선생님/구성원 초대는 다음 관리 기능 이후 진행
- 운영 보드의 선택 학생 주간 시간표는 완료
- 전체 학원 기준 주간 시간표와 날짜별 예외 일정은 후속 범위

## 현재 중요한 결정

- 완전한 캘린더 앱을 만들지 않습니다.
- 먼저 학생별 주간 반복 스케줄만 만듭니다.
- 외부 일정은 학생별 보강 불가 시간으로 취급합니다.
- 날짜별 예외, 드래그 앤 드롭, 학부모 캘린더 공유는 파일럿 검증 후 판단합니다.
